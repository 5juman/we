## 35. 블록체인 자산의 장외거래

## 43. 블록체인 기술의 발전사
* 블록체인 개념 없었음(bitcoin: 비트코인 / Bitcoin:기술, 현재의 블록체인 기술을 말함)
* 2015년 이코노미스트지 <세계를 재편할 블록체인 기술> 발표 => 블록체인 기술은 전세계적으로 핀테크 광풍을 일으킴
  ![image](https://github.com/5juman/we/assets/169249800/bf3df392-ff48-40d9-b073-454d67aaceb0)
* 세계의 대형 금융기관 & 은행이 블록체인을 연구
* 2017년 9월 중국 정부 <우리 블록체인 산업이 세계 선두가 되기를 바라며> 글로 블록체인 발전을 공개 지지, 13억 중국인에게 블록체인 기술 알림
* 블록체인은 금융, 보험, 소매업, 공증 등 실물경제 영역에서 빠르게 응용중

## 44. 신용을 제조하는 기기, 블록체인
* 블록체인 = 비대칭 암호 기술, 타임 스탬프, 합의 매커니즘 등 기술의 집대성
* 비트코인 => 비트코인 하나가 두 번 지불될 수 없음(=**이중 지불X**), 모든 탈중앙화 네트워크의 노드 일치를 보장(=**합의 매커니즘**, *노드란? 블록체인 네트워크 사용자)

  - 비대칭 암호 기술의 쓰임 ; 개인키의 보안
  - 타임 스탬프 ; 블록을 순서대로 연결
  - 작업 증명 매커니즘 ; 2,100만 개 비트코인의 공정 발행 문제 해결

    결론 : 블록체인 기술은 익명성, 탈중앙화, 투명성 등의 특징으로 '신용을 제조하는 기기'라는 칭호 얻음
    
## 46. 블록체인에 기록되는 정보는?
    블록체인 = 비트코인 네트워크의 장부
    비트코인 = 방부 속 페이지

블록헤더, 거래내역, 거래 계수기, 블록 크기 등

* 블록헤더 : 거래 정보 이외의 모든 정보
  
* 지난 **해시값** : 블록이 순서대로 연결된 것을 보증 (설명 : 각각의 블록마다 해시값이 있는데, a1-a2-a3 블록이라고 가정한다면, a2 블록에는 지난 블록의 해시값 즉 a1 블록의 해시값으로 블록을 증명함. 따라서, 해시값은 블록이 순서대로 연결된 것을 보증함.)
  
* **타임 스탬프** : 해당 블록이 생성된 시간을 기록
  
* 랜덤값 : 전 세계 채굴자들이 경쟁하여 계산한 답안
* 난이도 목표 : 해당 산술 문제의 난도 계수 점수
* 거래 내역에는 모든 거래 쌍방, 금액, 디지털 서명이 기록됨
* 거래 계수기 : 각 블록에 있는 거래의 수량을 표시
* 블록 크기 : 각 블록 데이터 크기를 표시

## 49. 블록체인의 종류

    블록체인의 종류는 퍼블릭, 프라이빗, 컨소시엄으로 나뉜다.

1. 퍼블릭 ; 공개형
   
   전세계 모든 개인/단체는 퍼블릭 블록체인을 통헤 거래 가능, 유효 확인 받기 가능
   
   ![image](https://github.com/5juman/we/assets/169249800/000e09b7-e736-47df-8f9e-2d35d41a2b27)
   * 모두 장부 기록 권한을 경쟁
     
   * 비트코인 블록체인이 대표적인 예
  
2. 컨소시엄 ; 반공개형
   
   어떤 단체/조직 내부에서 사용하는 블록체인
   
   사전에 몇 개의 노드를 장부 기록자로 지정
   
   모든 블록의 생성의 결정은 사전에 지정된 사람이 공동으로 결정
   
   * 다른 노드 : 거래 가능, 장부 기록 권한은 없음
   ![image](https://github.com/5juman/we/assets/169249800/d0924e5a-2c2c-4f09-9715-d34297960993)
   ![image](https://github.com/5juman/we/assets/169249800/3f370b79-91c4-4251-9d7e-07ecdd0f6143)

4. 프라이빗 ; 완전 폐쇄형
   
   블록체인 기술로 기록할 뿐, 장부 기록 권한이 비공개적
   
   내부 거래만 기록, 회사나 개인이 독점

## 53. 블록체인 자산의 특징(4) - 복제 불가
인터넷 방식 : 카피 방식 (예: 나의 핸드폰 사진을 다른 사람에게 공유하면 복제되어 공유된 것)

* 기존 : 판권 보유자에게 카피 복제(텍스트, 사진, 영상)은 늘 난제
* 블록체인: 블록체인이 자산인 것은 복제 불가이기 때문(카피가 아니라 암호화 기술로 전달)

* 기존 : 나의 핸드폰에 사진이 남아있음
* 블록체인 : A가 B에게 1000원 이체 => 나의 계좌에는 1000원이 남아있지 않음
